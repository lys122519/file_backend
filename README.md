## 一个经典的管理系统（很杂）
## 🔧 技术栈
### 前端
- 框架：Vue2

- 组件库：ElementUI 2
- 请求：Axios
- 路由：Vue Router
- 状态管理：Vuex

### 后端
- Java 8
- 开发框架：SpringBoot 2.x
- 数据访问：MyBatis + MyBatis Plus
- 项目管理：Maven
- 中间件：Redis
- 接口文档：Swagger （+ Knife4j）


### 存储
- 数据库：MySQL8.0

- 对象存储：HuaWei OSS


### 功能
- 登录/注册
  - 用户名+密码 登录
  - 邮箱+验证码 登录
  - 邮箱+验证码 找回密码
- 数据统计
- 用户管理
- 角色管理
- 菜单管理
- 文件管理
- 课程管理
- 文章管理
- 
- 视频播放
- 文章
